# 方案二色系统一规范

适用对象：本科毕业设计答辩 PPT、PPT 嵌图、论文输出图的展示版副本。  
原则：原图不覆盖，所有重配色图片单独存放；图表数值、坐标、标题和地图内容不做 AI 改写。

## 色系

| 用途 | 颜色 | 色值 |
|---|---|---|
| 主色 | 清华紫 | `#500070` |
| 强调深色 | 深紫 | `#2F0A42` |
| 辅助底色 | 淡雅紫 | `#E8D8EC` |
| 页面纸白 | 纸白 | `#FAF8F4` |
| 地图底色 | 蓝灰 | `#E0E8F0` |
| 浅底色 | 淡蓝 | `#E8F0F8` |
| 图表分类 | 地图蓝 | `#6F89AA` |
| 图表分类 | 灰绿 | `#688878` |
| 正文/坐标轴 | 墨灰 | `#2B2A2F` |
| 点缀色 | 学术金 | `#B88A3B` |

## 页面使用

1. 地图、流程、图表页以方案二为准：清华紫做标题和关键线索，地图蓝/灰绿/学术金承担分类，不使用高饱和红蓝撞色。
2. 纯文字和结论页参考方案三：更大留白、更少装饰、更克制的纸白底和段落层级。
3. 所有 PPT 页面仍以整页 PNG 插入，正文不保留可编辑文本框。
