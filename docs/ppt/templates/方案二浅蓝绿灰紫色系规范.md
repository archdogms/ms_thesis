# 方案二浅蓝绿灰色系规范

本版用于替代上一套含金色/玫红/内容紫色的配色。清华紫只作为模板识别色，用于校徽、页眉、页码、封面侧线和少量制度化强调；地图、热力图、数据图、知识图谱、流程图等内容图不使用紫色，统一采用灰绿、灰蓝、青灰与墨灰。原始图片不覆盖，所有图片均为新建副本。

## 核心色

| 用途 | 色名 | 色值 |
|---|---|---|
| 模板识别 | 清华紫 | `#5A0A6A` |
| 内容深色 | 深灰蓝 | `#4F7280` |
| 内容辅助 | 浅青灰 | `#7895A1` |
| 页面底色 | 纸白 | `#FAFCFC` |
| 地图底色 | 浅蓝灰 | `#DCEAF0` |
| 大面积浅底 | 雾蓝 | `#EAF2F5` |
| 图表分类 | 地图蓝绿 | `#6F9AAA` |
| 图表分类 | 灰绿 | `#7C9A88` |
| 深绿强调 | 深灰绿 | `#5E7E6A` |
| 正文/坐标轴 | 墨灰 | `#2B2F33` |

## 使用原则

1. 地图、图表、知识图谱、流程图统一折叠到蓝、绿、灰三类内容色。
2. 不使用红、橙、黄、金色，不使用高饱和商业渐变。
3. 地图底图保持浅蓝灰，边界和网格用低对比灰线，核心数据用灰绿、灰蓝、青灰强调，严禁用紫色表示地图/热力/类别数据。
4. 清华紫只保留在模板系统中，不进入地图图例、热力色带、柱状图、矩阵和流程节点。
5. 纯文字页继续保持方案三式留白，但色彩不跳出本版色系。
