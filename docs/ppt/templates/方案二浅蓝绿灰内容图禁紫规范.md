# 方案二浅蓝绿灰内容图禁紫规范

清华紫仅作为模板识别色，用于校徽、页眉、页码、封面侧线和少量制度化强调。

地图、热力图、数据图、知识图谱、流程图等内容图不使用紫色，统一采用灰绿、灰蓝、青灰与墨灰。

## 内容图色系

| 用途 | 色值 |
|---|---|
| 浅蓝灰底图 | `#DCEAF0` |
| 雾蓝浅底 | `#EAF2F5` |
| 地图蓝绿 | `#6F9AAA` |
| 深灰蓝 | `#4F7280` |
| 灰绿 | `#7C9A88` |
| 深灰绿 | `#5E7E6A` |
| 墨灰文字 | `#2B2F33` |
| 低对比线 | `#CBD6DA` |
