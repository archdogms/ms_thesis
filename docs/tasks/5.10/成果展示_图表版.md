## 六、核心结论

| 结论 | 关键数据 |
|---|---|
| 错位普遍存在 | 沉睡 28.5% + 空心 32.1% = 60.6% 载体存在错位 |
| 统计支撑稳固 | CMI×MI=−0.495, OAI×MI=−0.440, POI×CMI≈0 |
| 图谱扩展有效 | 0→1 跳：沉睡潜力 99→170，显化隐性文化网络 |
| 评级有带动效应 | 评级×评论量 r=0.546，评级×评分 r=0.522 |
| 东旅西文格局 | Material_Count×POI_Total=−0.475 |
| 潜力主廊道 | 西樵—九江—丹灶 147 个沉睡网格，占全区 86.5% |

**规划建议方向**：西南廊道做文化转化（评级申报 + 传习所 + 主题线路）| 东北都市圈做文化注入（非遗快闪 + 文化市集 + 解说系统）
