# 5.13 导师意见后续修改计划（重规划版）

> 本计划根据 `docs/tasks/5.13/5.13建议.md` 与论文初稿重新整理。  
> 本轮采用“必要口径修正 + 方法补强 + 研究不足说明”的策略，不推翻既有模型，但补做权重敏感性检验、评论文化识别度抽样和优先梯队清单。

## 一、核心判断

导师意见的关键不是单独修改某一张表，而是要求论文把“文旅融合程度”与“文旅融合潜力”区分清楚。现有数据能够支撑文化记忆、官方认证和旅游热度之间的相对关系判断，但还不能完整证明游客是否真正感知到文化内容，也不能直接证明某个点已经形成成熟的文旅融合状态。

因此，后续正文的总口径调整为：

```text
文化基础强 + 旅游热度高
        不是直接等于文旅融合成熟
        而是说明具备优先深化的文旅融合潜力
```

论文中“潜力”的操作化理解调整为：在已有文化基础、官方认证、旅游热度和空间条件的约束下，通过交通、解说、展示、活动、线路和运营等干预，进一步把文化资源转化为游客可感知体验的可能性。

## 四、暂不强行推进的任务

全量评论语义识别、CRI 指标构建、重新建立四指标模型、全量聚类分析等任务工作量较大，容易导致论文后期结构失控。本轮只做关键词抽样与稳健性检验，不把评论文化识别度扩展为新的主指标；全量语义分类、情感判断和主题模型仍放入研究展望。

## 五、答辩和后续说明口径

如果被追问“为什么不直接评价文旅融合深度”，可说明：本文当前数据更适合识别文化基础、官方认证和旅游热度之间的错位关系。游客是否真正识别文化、文化是否已经转化为体验产品，还需要更细的评论语义、问卷或现场调查支撑，因此本文将结果定位为“潜力识别”和“优先级诊断”，而不是成熟度评价。

如果被追问“权重是否主观”，可说明：本科论文阶段优先保证指标透明和可复核，采用等权与预设权重作为初步口径；本文已补做 CMI/OAI 与 THI 的权重敏感性检验，四组替代口径下分类一致比例均在 89.7% 以上，说明主要判断具有一定稳定性，但后续仍可通过专家打分、游客问卷或多元模型继续校准。
