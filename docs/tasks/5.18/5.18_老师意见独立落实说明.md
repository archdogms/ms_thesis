# 5.18 老师意见独立落实说明

## 一、处理边界

本文件只处理老师关于“旅游热度、评论文化识别度、潜力定义、梯队划分和三类数据关系”的意见，不修改论文正文、不修改 `ms_thuthesis` 中的 LaTeX 文件，也不改动原始数据和源数据。这里形成的是一个独立任务包，后续如果需要并入论文，可以再从本目录中选择文字和表格。

本次只使用已经存在的派生结果表：

- `final_comment_cultural_recognition.csv`
- `final_priority_ladder.csv`
- `final_sensitivity_summary.csv`

因此，本任务包更适合被理解为“基于现有结果的补充解释与可并入文本”，不是一次新的数据重算。

## 二、老师意见的核心问题

老师指出的问题可以归纳为四层：

1. 旅游侧现在主要用了 POI 数量、评分和评论量，反映的是“热度”，但不能直接判断游客是否识别到文化内容。
2. 高文化和高旅游不能被直接解释为“文旅融合已经成熟”。更合适的解释是：这些点具有更高的文旅融合深化潜力。
3. 有文化资源但旅游热度低的点，也不能简单说“潜力不行”。它们可能是外部条件不足、交通和运营不足，属于需要条件改善的潜力。
4. 文化、官方认证和旅游三个维度应该尽量分开观察，不能只把官方和旅游加在一起之后再和文化比较。

## 三、已经可以完成的内容

### 1. 将“旅游热度”降格为旅游侧可见度

现有 THI 仍然可以保留，但应明确它反映的是平台数据中的旅游可见度，而不是文旅融合深度。也就是说，THI 高只能说明某一载体周边 POI、评分或评论更强，不能说明游客已经围绕地方文化形成了明确认知。

因此，后续表述中应避免“高文化 + 高旅游 = 融合成熟”，改为“高文化 + 高旅游 = 优先深化潜力较高”。

### 2. 用评论文化识别度作为补充校验

在不重算原始评论数据的前提下，可以使用既有的 `final_comment_cultural_recognition.csv`，把评论中的文化关键词占比作为轻量校验。这个校验不能替代完整语义模型，但可以回答一个关键问题：游客评论中是否已经出现历史人物、非遗技艺、村落记忆、地方文化、博物馆、岭南、龙舟、醒狮、黄飞鸿、康有为等文化关键词。

从现有结果看，核心耦合区样本中，云泉仙馆、九江吴家大院、平洲传统玉器制作技艺等已经出现 30% 左右的文化关键词评论比例，但仍没有达到可以证明“融合成熟”的程度。部分空心景点区样本也有较高文化关键词比例，但其关键词往往依托西樵山、博物馆或周边文化环境，仍需要进一步把具体载体自身的故事讲清楚。

### 3. 把潜力重新分为三类

根据老师意见，现有“核心耦合区、沉睡潜力区、空心景点区”可以转化为规划语言中的三个梯队：

- 第一梯队：优先深化型。文化基础和旅游热度同时较高，适合作为品牌节点、线路节点和深度体验节点。
- 第二梯队：条件改善型。文化记忆和官方认证较强，但旅游热度偏低，需要补交通、展示、服务、活动和线上可见度。
- 第三梯队：文化补强型。旅游热度较高，但文化记忆或官方支撑偏弱，需要补地方叙事、非遗活动、解说系统和可被游客识别的文化内容。

这样处理之后，可以避免把“沉睡潜力区”写成低价值，也可以避免把“核心耦合区”写成已经成熟。

### 4. 文化、官方、旅游三维关系可以单独说明

现有结果已经可以支持三维解释：

- CMI 表示典籍和知识图谱中的文化记忆强度。
- OAI 表示官方认证和制度性识别程度。
- THI 表示平台数据中的旅游可见度。

三者不是相互替代关系。CMI 高而 THI 低，说明文化资源尚未转化为稳定旅游热度；THI 高而 CMI/OAI 低，说明已有客流和场地基础，但地方文化叙事需要补强；三者同时高，说明该载体适合作为优先深化对象，但仍需通过评论语义、产品内容和现场体验继续判断其文旅融合深度。

## 四、仍然没有完成的完整模型

老师提出的最完整做法，是把评论语义正式纳入旅游侧，构造类似“文化识别型旅游指数”或“文旅融合深度指数”。这一步需要全量评论语义分类、主题模型或人工标注校验，当前任务包没有做。

原因有三点：

1. 用户明确要求不改原文和源数据，本任务包不重新处理评论原始数据。
2. 现有评论关键词表只是抽样或轻量关键词识别，不能直接替代完整语义模型。
3. 如果把评论文化识别度纳入主模型，会改变 THI、MI 和分区结果，需要重新生成图表和论文叙述，不适合在不改正文的前提下完成。

因此，本任务包采用“轻量补充 + 口径规避”的方式完成老师意见：把 THI 明确限定为旅游热度，把评论文化识别度作为补充校验，把核心耦合区改写为优先深化潜力，而不是融合成熟。
